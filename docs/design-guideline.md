# 공공저작물 권리유형 자동분류 — UI/UX 디자인 가이드라인

> 대상: **한국문화정보원(공공누리 운영기관) 관리자**가 계약서·저작물을 검토하고 유형·권리 판정을 **확인·확정**하는 콘솔.
> 원칙: **판정 콘솔(Adjudication Console)** — 신뢰 · 정확 · 근거. 정보 밀도는 높되 위계는 명확. "실제로 동작하는 시스템"임이 화면에서 느껴져야 한다(처리 상태·신뢰도·근거 연결).

## 2. 타이포그래피
- **서체: Pretendard** (한국 공공/행정 UI 표준, KRDS 계열). 전 UI 통일.
- **숫자(신뢰도·건수)**: `tabular-nums` 고정폭.
- **타입 스케일**
  | 역할 | 크기/굵기 | 용도 |
  |---|---|---|
  | Display | 20 / 700 | 페이지 타이틀 |
  | Section | 15 / 700 | 카드/섹션 제목 |
  | Body | 14 / 400 | 본문·표 |
  | Label | 13 / 500 | 필드 라벨·탭 |
  | Caption | 12 / 400 | 보조·단위 |
- 행간 1.5, 문장은 sentence case(한글), 라벨은 명사형.

---

## 3. 레이아웃 · 셰이프
- 컨텐츠 폭: 상세 `max-w-5xl`, 목록 풀폭 카드그리드(`sm:2 / lg:3 / xl:4`).
- 라운드: 카드 `12px`, 입력 `8px`, 칩 `999px`.
- 그림자: 카드 `0 1px 3px rgba(15,23,42,.06)`, hover `0 6px 16px rgba(15,23,42,.10)`.
- 간격: 섹션 간 `24px`, 카드 패딩 `20~24px`.

---

## 4. 컴포넌트 정의 (3메뉴 공통)
- **PageHeader** — 아이콘 + 타이틀(Display) + 한 줄 설명(Caption) + 우측 액션.
- **SectionCard** — 표면·경계·라운드·그림자, 헤더(Section) + 본문. 모든 결과 블록의 기본 그릇.
- **StatusBadge** — 점(dot)+라벨, 판정 semantic 색. (허용/금지/확인필요/없음)
- **ConfidenceMeter** — 신뢰도 막대(색=판정) + `%`(tabular). 숫자에 의미를 부여해 "동작 체감".
- **RightsTable** — 권리 · 판정(Badge) · 신뢰도(Meter) · 근거번호. 행 hover, 반응형 가로 스크롤.
- **MetaList** — 라벨(한글 사전 매핑)+값. **내부 키(pattern_detected, extraction_confidence, checkbox_info 등)는 숨김.** 관리자가 아는 용어만 노출.
- **EvidenceHighlight (시그니처)** — 원문 위에 판정 근거를 색·라벨로 표시. 판정↔문서 연결로 **근거·책임성**을 보여줌.
- **ProcessStepper** — 업로드 → OCR·메타데이터 → 유형 → 권리 진행 단계 표시(처리중 실시간).
- **ResultSummary** — 허용/확인필요/없음/근거 요약 밴드(상단 고정).
- **EmptyState / ErrorState** — 무엇을·왜·다음 행동을 콘솔 어조로 안내.

---

## 5. 시그니처
**근거 기반 판정(Evidence-grounded verdict)** — 모든 유형·권리 판정이 **원문 근거와 색으로 연결**된다. 관리자가 "왜 이 판정인가"를 문서에서 즉시 검증 → 자동분류를 신뢰·확정할 수 있는 콘솔.

## 6. 관리자 카피 원칙
- 사용자가 통제·인지하는 말로 명명(저작물명·저작권자·이용기간…), 시스템 내부 용어 금지.
- 액션은 결과와 동일 어휘(실행→실행됨). 에러는 사과 대신 원인+다음 행동.
- 빈 화면은 다음 행동으로 안내.

---

## 7. 적용 순서 (이 가이드 기반)
1. 전역: Pretendard 도입 + 토큰(색/타이포) 정리(globals.css, tailwind.config).
2. 공통 컴포넌트화: PageHeader / SectionCard / StatusBadge / ConfidenceMeter / MetaList / EvidenceHighlight / ProcessStepper.
3. 3메뉴(검사하기·권리추정·통합) 결과·목록·업로드 화면에 컴포넌트 일괄 적용 → 타이포·간격·색 통일.
4. 메타데이터 라벨 사전 정비(내부키 숨김), 처리중 상태 표시 추가.
