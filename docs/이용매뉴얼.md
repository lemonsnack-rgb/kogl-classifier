# 공공저작물 권리유형 자동분류 서비스 — 이용 매뉴얼

> 본 매뉴얼은 서비스의 **사용 방법**과 **결과 화면을 보는 방법**을 안내한다.
> 서비스 주소: https://kogl-classifier.vercel.app

---

## 1. 서비스 개요

계약서·저작물 파일을 업로드하면 **공공누리 유형(제0~4유형)** 과 **AI 학습 활용 여부**, **저작권 권리 판정(허용/금지)** 을 자동으로 분류·추정하고, 그 **판정 근거를 원문에서** 확인할 수 있는 서비스다.

| 메뉴 | 무엇을 하나 | 결과로 무엇을 보나 |
|---|---|---|
| **검사하기** | 계약서·저작물로 공공누리 유형 분류 | 공공누리 유형 + 저작물별 유형·AI 판정 + 메타데이터 |
| **권리추정** | 계약서에서 권리 항목별 허용/금지 추정 | 권리 판정표 + 근거 |
| **통합검사** | 위 둘을 한 화면에서 제공 | 메타데이터 + 유형·AI + 권리 + 근거 하이라이트 |

> **자동 판정 + 사람 보정 원칙**: 모든 판정(유형·AI·권리)은 **자동으로 판정값을 먼저 제시**하고, 담당자가 **[수정]으로 확정·보정**한다. 최초 자동판정은 보존되며, 사람이 바꾸면 그 이력(사유 포함)이 누적된다.

---

## 2. 시작하기

1. 브라우저에서 https://kogl-classifier.vercel.app 접속
2. **이메일 · 비밀번호** 입력 후 **[로그인]**
3. 좌측 사이드바에서 **검사하기 · 권리추정 · 통합검사** 중 원하는 메뉴 선택

---

## 3. 기본 사용 흐름 (공통)

세 메뉴 모두 사용법은 동일하고 간단하다.

1. **목록 화면**에서 **[새 검사 시작]**(또는 새 통합 검사/새 권리추정) 카드를 누른다.
2. **파일을 업로드**하고(권리추정은 텍스트 붙여넣기도 가능) 검사 명칭을 입력한다.
3. **[실행]** 을 누르면 검사가 **접수되고 목록 화면으로 돌아온다**(접수 안내 메시지 표시).
4. 처리는 **백그라운드로 진행**되며, 목록 카드의 상태가 처리중 → 완료로 **자동 갱신**된다. 완료된 카드를 누르면 결과를 확인한다.

### 처리 진행 단계

```
업로드 → OCR·메타데이터 → (공공누리 유형·AI 판정) → 권리 판정 → 완료
```

- 처리 중에도 다른 작업을 계속할 수 있다. 목록은 자동으로 갱신된다(별도 새로고침 불필요).
- 문서 크기에 따라 수십 초~수 분 걸릴 수 있다. **처리 시간이 길어도 오류가 아니며**, 완료되면 상태가 자동으로 바뀐다.
- 실패 시 상태가 "실패"로 표시된다 → 파일을 확인하고 다시 시도한다.

### 목록 카드 보는 법 (공통)

- **제목**: 검사 명칭(또는 파일명)
- **배지**: 공공누리 유형(제N유형) 또는 권리 요약(허용 N / 확인필요 N)
- **상태**: 완료 · 처리중 · 실패
- **하단**: 저작물/근거 건수 · 등록자 · 날짜

상단 **검색창**에 검사명·파일명을 입력해 원하는 건을 빠르게 찾을 수 있다.

## 4. 상세 화면 공통 구조 — 좌우 2단 콘솔

**검사하기와 통합검사는 동일한 상세 화면 구조**를 사용한다. 카드를 클릭하면 화면이 **좌우 2단**으로 열린다.

### 좌측 — 정보 + 선택 목록
- 상단 **"계약서·저작물" 목록**: 여기서 항목을 **클릭하면 우측에 상세가 열린다.**
  - **[계약서 추출 메타데이터]** → 우측에 계약서에서 추출한 메타데이터
  - **저작물 목록(1, 2, 3 …)** → 클릭한 저작물의 메타데이터·판정
  - (각 항목 우측의 **> 화살표**와 "조회" 표시가 클릭 가능함을 나타낸다.)
- 그 아래로 계약서 기본정보·공공누리 자동분류 정보(대표 유형)·근거(검사하기), 또는 유형·권리 판정·근거 하이라이트(통합검사)가 표시된다.

### 우측 — 공유 상세 패널
좌측에서 무엇을 클릭했는지에 따라 내용이 바뀐다.
- **계약서**를 클릭 → 계약서 추출 메타데이터(항목/값)
- **저작물**을 클릭 → 그 저작물의 **공공누리 유형 판정 + 20항목 메타데이터**

### 저작물 "공공누리 유형 판정" 배너
저작물을 선택하면 우측 상단에 판정 배너가 나온다.
- **유형 배지**: 제0~4유형 (예: `제3유형 · 출처표시 + 변경금지`)
- **AI 배지**(제1~4유형에만 병기): `AI 활용 가능` / `AI 활용 불가` / `AI 판단 불가` — **제0유형은 AI 해당 없음**
- **자동 추정 · 확인 권장**: 자동 판정 신호가 약할 때 표시(담당자 확인 권장)
- **사람 수정됨(자동판정과 다름)**: 담당자가 최초 자동판정을 바꾼 경우 표시

> **저작물 메타데이터 20항목**: 저작물명 · 유형 · 디지털화형태 · 설명 · 주제어 · 언어 · 제작일 · 제작자 · 저작권자 · 공동저작자 · 저작인접권자 · 공개유형 · 저작물성 · 비보호저작물 · 업무상저작물 · 상업적이용허락 · 저작재산권 · 공동저작자동의 · 유효기간 · 초상권

## 5. 판정 수정하기 (유형 · AI · 메타데이터)

자동 판정값을 담당자가 보정할 수 있다. **본인이 등록한 검사만 수정 가능**하며, 타인 검사에는 "본인 검사만 수정 가능합니다" 안내가 표시된다.

1. 좌측 목록에서 **저작물을 선택**한다.
2. 우측 패널 우상단 **[수정]** 을 누른다.
3. "공공누리 유형 판정"에서 값을 고른다.
   - **공공누리 유형**: 미판정 / 제0~4유형
   - **AI 학습 활용**: 활용 가능 / 활용 불가 / 판단 불가 (제0유형 선택 시 자동 비활성)
   - **수정 사유**(선택): 왜 바꾸는지 근거를 적는다. 예) "계약서 제6조에 상업적 이용 명시"
   - 아래 20항목 메타데이터도 이 편집 화면에서 직접 채우거나 수정할 수 있다(빈 항목은 빈 입력칸으로 표시).
4. **[저장]** 을 누른다.

> **왜 사유를 적나?** 최초 자동판정과 사람 확정값, 그리고 사유가 **이력으로 누적**되어 향후 **판정 모델의 품질 개선(재학습·규칙 보정)** 근거로 활용된다. 즉 담당자가 쓰면서 자연스럽게 라벨링이 이뤄진다.

## 6. 권리추정 — 결과 보는 방법

상세 화면은 위에서 아래로 다음과 같이 구성된다.

1. **판정 요약 밴드**: `허용/확정` · `확인필요` · `없음` · `근거` 건수를 한눈에 보여준다.
2. **권리 그룹 테이블**: 권리를 그룹(저작재산권 · 이용조건 · 계약성격 · 이용범위 · 대가조건)별로 묶어 표시한다.
   - **권리**: 복제권·전시권·상업적 이용 등
   - **판정**: 허용 · 금지 · 확인필요 · `-`(없음)
   - **신뢰도**: 판정을 확신하는 정도(%)
   - **근거번호**: 판정의 근거가 된 문장 번호

> 처리 중이면 진행 단계(스텝퍼)가 먼저 보이고, 완료되면 자동으로 결과로 바뀐다.

## 7. 통합검사 — 결과 보는 방법

계약서와 저작물을 함께 올리면 **한 화면(§4 좌우 2단 콘솔)** 에서 메타데이터·유형·AI·권리·근거를 모두 확인할 수 있다.

- **좌측**: 공공누리 유형(유형분류 모델) · 권리 판정 요약 · **원본 문서(근거 하이라이트)** · 권리 그룹 테이블, 그리고 상단의 **계약서·저작물 목록**.
- **우측(공유 패널)**: 계약서 클릭 시 계약서 메타데이터 / 저작물 클릭 시 그 저작물의 유형·AI 판정 + 20항목.

**원본 문서 (근거 하이라이트)** — OCR 원문 위에 판정 근거를 **색으로 표시**한다.
- 🔵 파랑 = 유형 근거 · 🟢 초록 = 권리 허용 · 🔴 빨강 = 권리 금지
- → "왜 이 판정인가"를 원문에서 바로 확인할 수 있다.

> 계약서 추출 메타데이터의 **항목은 문서 유형에 따라 다르다**(고정 목록이 아님). 계약서마다 뽑히는 필드가 다를 수 있다.

## 8. 결과 이해하기 (공통)

### 8.1 공공누리 유형

| 유형 | 의미 |
|---|---|
| **제0유형** | **자유이용** (비보호·만료 저작물, 초상권 없는 업무상저작물 등) |
| 제1유형 | 출처표시 |
| 제2유형 | 출처표시 + 상업적 이용금지 |
| 제3유형 | 출처표시 + 변경금지 |
| 제4유형 | 출처표시 + 상업적 이용금지 + 변경금지 |

### 8.2 AI 학습 활용 여부 (AI유형)

제1~4유형에 **병행**해서 해당 저작물을 AI 학습에 활용할 수 있는지 표시한다.

| 표시 | 의미 |
|---|---|
| AI 활용 가능 | 권리자 동의 등 근거로 AI 학습 활용 가능 |
| AI 활용 불가 | 미동의 등으로 활용 불가 |
| AI 판단 불가 | 근거가 없어 판단 보류(담당자 확인 권장) |

> **제0유형(자유이용)은 AI유형 해당 없음**(자유이용에 AI 학습 포함).

### 8.3 권리 판정 상태

| 표시 | 의미 |
|---|---|
| 허용 | 해당 권리 이용이 허용됨 |
| 금지 | 해당 권리 이용이 금지됨 |
| 확인필요 | 자동 판정이 모호하여 담당자 확인을 권장 |
| - (없음) | 계약서에서 근거를 찾지 못함(미식별) |

### 8.4 신뢰도와 근거

- **신뢰도(%)**: 모델이 해당 판정을 확신하는 정도(0~100%). 낮으면 **근거 문장을 함께 확인**한다.
- **근거번호 · 하이라이트**: 판정과 연결된 근거의 위치. 통합검사에서는 원문에 색으로 표시되어 즉시 대조할 수 있다.
- **확인필요 · 자동 추정** 항목은 자동 판정을 그대로 신뢰하기보다 원문 근거를 보고 담당자가 최종 판단한다.

## 9. 자주 묻는 질문 (FAQ)

**Q. 목록에 기존 검사 결과가 보이지 않습니다.**
A. 계정별로 접근 범위가 다릅니다. 담당 계정으로 로그인했는지 확인하세요.

**Q. 저작물 [수정] 버튼이 없고 "본인 검사만 수정 가능합니다"라고 나옵니다.**
A. 판정 수정은 **검사를 등록한 본인 계정에서만** 가능합니다. 본인 검사인지 확인하세요.

**Q. 유형·AI를 바꿨는데 나중에 최초 자동판정을 알 수 있나요?**
A. 네. 최초 자동판정은 보존되며, 값이 바뀐 저작물은 배너에 **"사람 수정됨(자동판정과 다름)"** 으로 표시됩니다(마우스를 올리면 최초 자동판정을 보여줍니다). 수정 이력(사유 포함)도 누적됩니다.

**Q. 제0유형은 무엇인가요?**
A. **자유이용** 저작물입니다. 비보호·보호기간 만료 저작물이거나, 초상권 문제가 없는 업무상저작물 등일 때 자동으로 제안됩니다.

**Q. AI 배지가 안 보이는 저작물이 있습니다.**
A. **제0유형**은 AI유형 해당이 없어 배지를 표시하지 않습니다. 제1~4유형이면 활용 가능/불가/판단 불가 중 하나가 항상 표시됩니다.

**Q. 계약서 메타데이터가 검사마다 항목이 달라 보입니다.**
A. 계약서 메타데이터는 **문서 유형에 따라 추출 항목이 달라지는** 구조입니다(고정 목록이 아님). 저작물 메타데이터(20항목)는 고정입니다.

**Q. 지원 파일 형식은?**
A. PDF, JPG, PNG, TIFF. 계약서는 PDF 권장.

**Q. "계약 제목"과 "저작물명"의 차이는?**
A. **계약 제목**은 계약서가 다루는 대상의 제목(계약서 추출 정보)이고, **저작물명**은 개별 저작물 각각의 이름(저작물 메타데이터)입니다.

**Q. 신뢰도가 낮은 판정은 어떻게 하나요?**
A. 원문 근거(하이라이트·근거 문장)를 확인해 담당자가 [수정]으로 최종 확정하세요. "확인필요"·"자동 추정" 항목도 동일합니다.
