# 공공누리 신유형(제0유형·AI유형) 판정 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** SSU 저작물 메타 20항목 + HMC 유형(1~4)을 조합하는 후처리 규칙으로 제0유형·AI유형을 저작물별로 판정·표시하고, 기존 [수정] 기능으로 보정 가능하게 한다.

**Architecture:** HF Pipeline(`pipeline-server/main.py`)에 순수함수 `resolve_kogl_type`를 추가해 통합(`/process-combined`)·검사하기(`/process`)에서 저작물별로 적용. 프론트는 KOGL-0 유형과 AI유형(직교 boolean)을 뱃지로 표시하고 편집 UI에 유형 드롭다운·AI 체크박스를 더한다.

**Tech Stack:** FastAPI + httpx + supabase-py (HF Space), Next.js 14 + TypeScript, Supabase(Postgres/JSONB).

## Global Constraints

- 판정 단위는 **저작물별**. 레코드 대표유형(계약서 HMC 결과)은 불변.
- **AI유형은 `KoglType` enum에 넣지 않는다.** 별도 boolean `ai_type_applied`. 제0유형(KOGL-0)은 AI유형 N/A(항상 false, 미표시).
- 자동판정값은 **기본 표시만** 하고 확정/기각 상태머신을 두지 않는다. 보정은 기존 [수정] 경로로 통일(전 유형 공통). 불확실은 `type_low_confidence` 표시로만 안내.
- 만료 판정은 날짜계산 없이 `non_protected_work` 신호 기반(YAGNI).
- HMC·권리 모델 및 `mapHMCToKoglType`은 **불변**.
- KOGL-0 색상 `#6B21A8`, label "제0유형", description "자유이용".
- 커밋 메시지 끝에 `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>` 추가.

- [ ] **Step 3: 함수 구현**

`pipeline-server/main.py`의 `map_ssu_to_work_fields` 정의 **아래**에 추가:
```python
_TRUE_TOKENS = ("예", "yes", "true", "해당", "포함", "있음", "적용")
_FALSE_TOKENS = ("아니", "no", "false", "미해당", "없음", "불포함", "미적용", "해당 없음", "해당없음")
_EXPIRED_TOKENS = ("만료", "비보호", "expired", "public domain", "퍼블릭도메인")


def _is_true(v) -> bool:
    if v is None:
        return False
    s = str(v).strip().lower()
    if not s:
        return False
    if any(t in s for t in _FALSE_TOKENS):
        return False
    return any(t in s for t in _TRUE_TOKENS)


def _is_expired(v) -> bool:
    if v is None:
        return False
    s = str(v).lower()
    return any(t in s for t in _EXPIRED_TOKENS)


def resolve_kogl_type(work_meta: dict, hmc_type):
    """SSU 저작물 메타 + HMC 유형 → 저작물별 신유형 판정(후처리 규칙)."""
    non_protected = work_meta.get("non_protected_work")
    work_for_hire = work_meta.get("work_for_hire")
    portrait = work_meta.get("portrait_rights")
    consent = work_meta.get("co_author_consent")

    hmc_valid = hmc_type if hmc_type in ("KOGL-1", "KOGL-2", "KOGL-3", "KOGL-4") else None
    resolved = hmc_valid
    ai_candidate = False
    reason = None
    low_confidence = False

    if _is_true(non_protected) or _is_expired(non_protected):
        resolved = "KOGL-0"
        reason = "비보호(만료)저작물"
    elif _is_true(work_for_hire) and not _is_true(portrait):
        resolved = "KOGL-0"
        reason = "업무상저작물, 초상권 없음"
    else:
        if hmc_valid is None:
            reason = "유형 신호 부족(계약서 판정 없음)"
        else:
            reason = "계약 기반 유형(상업·변경 축)"
        if _is_true(consent):
            ai_candidate = True

    if resolved == "KOGL-0":
        ai_candidate = False

    if not (_is_true(non_protected) or _is_expired(non_protected) or _is_true(work_for_hire) or hmc_valid):
        low_confidence = True

    return {
        "resolved_type": resolved,
        "ai_candidate": ai_candidate,
        "reason": reason,
        "low_confidence": low_confidence,
    }
```

    # ── 저작물별 SSU 추출 + 신유형 판정 (검사하기) ──
    if sb:
        try:
            wq = sb.table("works").select("id, work_file_url, work_filename").eq("contract_id", contract_id).execute()
            work_rows = wq.data or []
            async with httpx.AsyncClient(timeout=600.0) as wclient:
                for wr in work_rows:
                    path = wr.get("work_file_url")
                    if not path:
                        continue
                    try:
                        blob = sb.storage.from_("works").download(path)
                        wssu = await _ssu_extract(wclient, blob, wr.get("work_filename") or "work", "기타문서")
                        wmeta = wssu.get("consolidated_metadata") or wssu.get("metadata") or {}
                        mapped = map_ssu_to_work_fields(wmeta)
                        verdict = resolve_kogl_type(mapped, kogl_type)
                        sb.table("works").update({
                            "work_name": mapped.get("work_name"),
                            "digital_format": mapped.get("digital_format"),
                            "description": mapped.get("description"),
                            "language": mapped.get("language"),
                            "created_date": mapped.get("created_date"),
                            "copyright_holder": mapped.get("copyright_holder"),
                            "resolved_type": verdict["resolved_type"],
                            "ai_type_applied": verdict["ai_candidate"],
                            "type_reason": verdict["reason"],
                            "type_low_confidence": verdict["low_confidence"],
                            "ocr_status": "completed",
                        }).eq("id", wr["id"]).execute()
                    except Exception as e:
                        sb.table("works").update({"ocr_status": "failed"}).eq("id", wr["id"]).execute()
        except Exception as e:
            print(f"works 신유형 판정 오류: {e}")
